JORAN POCKET

手机里的 Claude Code
——躺着就能指挥。

在你 Mac 终端里跑 pocket， 手机 Safari 打开蹦出的网址， 终端实时投屏，compose 栏打字、大按钮点 y/n。 WebRTC P2P 直连，没有中间服务器看到你的终端内容。

零账号

终端用户不注册任何东西。下二进制，开终端，扫码。

零 VPN 冲突

不用 Tailscale / WireGuard，不占手机 VPN 槽位，Astrill 等 VPN 可以同时跑。

端到端加密

WebRTC DataChannel 自带 DTLS 加密；信令服务器看不到你的终端字节。

P2P 直连

ICE 打洞走 STUN；家用网络下通常 <10ms 延迟。

三步上手

  1. 装 helper（Mac 一次性）： 
    curl -fsSL https://joran-pocket.pages.dev/install.sh | bash
  2. 每次用时，Mac 终端： 
    pocket
    终端里会打印 6 位码，比如 EK7P2M
  3. iPhone Safari 打开 https://joran-pocket.pages.dev/p/EK7P2M →WebRTC 握手→终端出现。分享→添加到主屏幕，下次一键进。

工作原理

Mac 上的 pocket helper 启一个 tmux 会话让 Claude Code 跑在里面。 helper 连信令服务器（Cloudflare Worker）注册一个 6 位一次性码。 手机 PWA 输入码→通过同一个 Worker 中转 SDP/ICE 握手信号。 握手完成后，终端字节流走 WebRTC DataChannel 点对点—— 中继服务器看不到，本机和手机之间直连。

技术栈

Go + pion/webrtc Cloudflare Workers React + xterm.js MIT OSS

自己部署

git clone https://github.com/cocohahaha/joran-pocket
cd joran-pocket
# 部署你自己的信令 Worker（免费额度内）
cd signaling && npx wrangler deploy
# 部署 PWA 到 Cloudflare Pages
cd ../pwa && npm run build && npx wrangler pages deploy dist
# 本地构建 helper（或直接 brew install，发布后提供）
cd ../helper && go build -o pocket .
